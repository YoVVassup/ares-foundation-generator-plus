**Ares Foundation Generator +** — это десктопное приложение для создания и редактирования фундаментов зданий для игровых уровней (например, ARES). Поддерживает ортогональную и изометрическую проекции, наложение шаблона, отмену/повтор действий, экспорт в INI и изображения.

![Заглушка скриншота](screenshot.png)

## Возможности

- Интерактивное редактирование сетки (ЛКМ – добавить, ПКМ – удалить)
- Режим контура (удерживайте `Ctrl` для редактирования контура фундамента)
- Автоматическая генерация контура по заполненным ячейкам
- Ортогональная и изометрическая проекции
- Наложение изображения-шаблона (PNG, JPG и др.) с прозрачностью и масштабом
- Отмена/Повтор (Ctrl+Z / Ctrl+Y)
- Экспорт данных фундамента в INI-файл (готово для ARES)
- Экспорт ортогонального и изометрического видов в PNG
- Загрузка существующих INI-файлов (перетаскиванием или через кнопку)
- Регулируемый размер сетки (от 5x5 до 50x50)
- Цветовые схемы и поддержка нескольких языков (English, Русский, 简体中文, 繁體中文)
- Изменяемый размер окна с зумом и панорамированием

## Требования

- Python 3.8+
- Pygame
- NumPy
- Tkinter (обычно входит в состав Python)

## Установка и запуск

1. Клонируйте репозиторий:
   ``` bash
   git clone https://github.com/YoVVassup/ares-foundation-generator-plus.git
   cd ares-foundation-generator-plus
   ```

2. (Опционально) Создайте и активируйте виртуальное окружение:
   ``` bash
   python -m venv .venv
   .venv\Scripts\activate   # Windows
   source .venv/bin/activate  # Linux/Mac
   ```

3. Установите зависимости:
   ``` bash
   pip install pygame numpy zstandard
   ```

4. Запустите приложение:
   ``` bash
   python main.py
   ```

## Сборка исполняемого файла (Windows)

Предоставлен PowerShell-скрипт `Run.ps1`. Он использует **Nuitka** для компиляции программы в standalone `.exe`.

- Установите Nuitka: `pip install nuitka`
- Запустите `.\Run.ps1` в PowerShell (может потребоваться изменение политики выполнения: `Set-ExecutionPolicy RemoteSigned -Scope Process`)

Результат будет в папке `Ares Foundation Generator Plus`.

## Основное использование

- **Левая кнопка мыши** – заполнить ячейку
- **Правая кнопка мыши** – очистить ячейку
- **Ctrl + ЛКМ/ПКМ** – редактировать ячейки контура
- **Средняя кнопка мыши** – панорамирование вида
- **Ctrl + Средняя кнопка** – панорамирование шаблона
- **Колесо мыши** – масштабирование сетки
- **Ctrl + Колесо мыши** – масштабирование шаблона
- **G** – сгенерировать контур по заполненным ячейкам
- **C** – очистить всю сетку
- **L** – загрузить INI-файл
- **Ctrl+E** – экспорт в INI
- **Ctrl+Shift+S** – сохранить изображения
- **F** – подогнать сетку под экран
- **P** – переключить проекцию
- **O** – отображать/скрывать координаты
- **R** – сбросить вид
- **Z** – сбросить масштаб
- **Ctrl+Z / Ctrl+Y** – отменить / повторить
- **F1** – показать/скрыть справку

## Структура файлов

- `main.py` – точка входа, главный цикл, обработка событий
- `grid.py` – модель данных сетки и генерация контура
- `renderer.py` – отрисовка ортогональной/изометрической проекций
- `ui.py` – компоненты интерфейса (кнопки, ползунки, диалоги)
- `commands.py` – паттерн «Команда» для отмены/повтора
- `localization.py` – поддержка нескольких языков
- `settings.py` – постоянные настройки (INI)
- `constants.py` – глобальные константы и цветовые схемы
- `utils.py` – вспомогательные функции для путей
- `language_*.ini` – файлы переводов
- `icons/` – файлы иконок (SVG/PNG)
- `Unifont.ttf` – файл шрифта (опционально)
